# Contributing to AI SQL Agent

Thank you for your interest in contributing to this project! We welcome contributions from the community to help improve this tool.

## How to Contribute

### 1. Reporting Bugs
- Search existing issues to see if the bug has already been reported.
- If not, open a new issue using the "Bug Report" template.
- Include clear steps to reproduce the bug and details about your environment.

### 2. Suggesting Enhancements
- Open a new issue using the "Feature Request" template.
- Clearly describe the proposed enhancement and why it would be useful.

### 3. Pull Requests
- Fork the repository and create a new branch for your changes.
- Ensure your code follows the project's style and is well-documented.
- Run any available tests before submitting.
- Provide a clear description of the changes in your pull request.

## Development Setup
- Follow the instructions in the `README.md` to set up your local development environment.
- Use `pip install -r requirements.txt` to install dependencies.

## Code of Conduct
Please be respectful and constructive in all interactions within this project.
